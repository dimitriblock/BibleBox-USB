
# Localized BibleBox Repo


## Installation
1. Follow instruction: http://biblebox.org/make-your-own/
2. SSH to root@biblebox (PW: biblebox)
3. Go to usb drive with `cd /mnt/usb/`
4. Run `chmod 755 install_h5ai.sh`
5. Run `sh install_h5ai.sh`
